# DUSTMAN - Plymouth Mod for My Winter Car

Adds a 1970 Plymouth to My Winter Car. Buy it at the repair shop, drive it around, upgrade it.

## Features

- Buyable at the repair shop for 56,900 MK
- GT upgrade package (38,600 MK) - swaps the body panels and bumps the engine to 288hp
- GT rims (7,600 MK)
- Working doors, trunk, handbrake, lights, ignition, gear shifter
- Dashboard with speedo and tachometer
- Deformable body panels
- Suspension impact sounds
- Radio

## Installation

1. Install MSCLoader if you haven't already
2. Drop the mod DLL and asset bundles into your `Mods` folder
3. Launch the game

## Building from source

Open `Plymouth.sln` in Visual Studio. You'll need references to the game assemblies and MSCLoader - check the `.csproj` for paths.
